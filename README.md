# 📦 FedEx Logistics EDA Project

## 📌 Problem Statement
Analyze supply chain data to identify inefficiencies in delivery performance and logistics cost.

## 🎯 Business Objective
- Reduce delivery delays  
- Optimize freight cost  
- Improve vendor performance  
- Increase customer satisfaction  

## 🛠 Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

## 📊 Key Insights
- Certain vendors show high delivery delays  
- Air shipment is faster but more expensive  
- High freight cost does not always guarantee faster delivery  
- Shipment mode and vendor selection impact delivery performance  

## 📈 Visualizations
- Correlation Heatmap  
- Lead Time Distribution  
- Freight Cost Analysis  
- Shipment Mode Comparison  

## 📌 Conclusion
Optimizing vendor selection and choosing the right shipment mode can significantly reduce delivery delays and costs, improving overall supply chain efficiency.
